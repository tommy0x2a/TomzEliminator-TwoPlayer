# TomzEliminator - Two Player

**Multiplayer version** of the classic 1981 Gremlin arcade game *Eliminator*.

- Only human players (no AI droids)
- Peer-to-peer multiplayer using WebRTC + PeerJS
- Up to 2 players (expandable)
- Same core gameplay: destroy the base station by shooting down the tunnel, drive enemies into the base, etc.

## Controls (Player 1)
- **D** = Turn Left
- **F** = Turn Right  
- **J** = Thrust
- **K** = Fire

## How to Play
1. One player creates a room
2. Other player joins with the room code
3. Both players control their own ships in real-time
